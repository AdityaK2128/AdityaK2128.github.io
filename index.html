ToDo List App

This ToDo list App aims to give users the minimal and the simplest experience for the functionality they want from the App
Task Creation: Users can easily add new tasks to the to-do list. This involves providing a title or description for the task.
Task Deletion: Users should be able to delete tasks they no longer want on their to-do list. This can be achieved through a delete button or swipe-to-delete gesture.
Task Update: Users may want to edit or update the details of a task. This includes changing the task title, adding notes, specifying due dates, or updating task priority.
Task Completion: Users should be able to mark tasks as completed. This often involves a checkbox or a "mark as done" button. Completed tasks are usually visually distinguished from unfinished ones.

TicTacToe Game

AI powered TicTacToe Game with 3 Difficulty levels of play
User Interface:A user-friendly interface for the Tic-Tac-Toe board. User can make their move by tapping on the board on their turn. Play Again button is also there if user wants to play the Game Again.
Game Logic: Whenever the User or the AI gets a pair of 3 same movel in a Straight Line then that user is declared as Winner Implement the rules for winning, losing, and drawing. Track the game state and determine when it's over.
AI Development: It has opponent for Easy, Medium, and Hard difficulty levels. For the Easy level, the AI can make random moves. For the Medium level, the AI can make slightly smarter moves, such as blocking the player from winning. For the Hard level, implement a more sophisticated AI algorithm, such as the minimax algorithm with alpha-beta pruning.
User Interaction: Allow the user to make their moves by tapping on the board. Update the game state and the UI after each move.
Alert Popups: Show alert popups when the game ends (win, lose, or draw). Customize the alerts to display the result and options to play again or exit the game.
